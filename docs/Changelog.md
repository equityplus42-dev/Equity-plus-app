# Project Changelog

## [1.0.0] - 2026-07-02
### Added
- Express.js backend folder template, controllers, services, repositories architecture.
- Self-referencing relational User models and materialized path tree HierarchyNodes in database.
- Zod validation middlewares and Bcrypt password encryption.
- Multi-level referral points distribution engine.
- Multer in-memory uploads with Cloudinary upload streams.
- Base64 QR code generators using native qr encoder.
- User Flutter Client App: Onboarding slides, glassmorphism dashboards, timeline lists, and recursive tree branches view.
- Admin Flutter Client App: Approvals queues, user directories switches, global tree views, settings variables dashboards, and analytics csv export simulator.
- Raw MySQL schema.sql DDL backups.
- Local assert HTTP integration test files.
